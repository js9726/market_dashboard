# Backend Script Launchers

Use these BAT files for local broker sync:

- `start_moomoo_bridge.bat` - starts the real MooMoo dashboard bridge.
- `start_ibkr_bridge.bat` - starts the IBKR dashboard bridge.
- `start_live_quote_daemon.bat` - old shortcut name; forwards to MooMoo bridge.

The actual bridge implementation is in `packages/dashboard-bridge`.
